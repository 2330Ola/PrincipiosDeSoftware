Tarea: Principios de Ingeniería de Software - Josue Ola

Este repositorio tiene tres ejemplos básicos en Python que muestran tres principios importantes de la ingeniería de software: DRY, KISS y YAGNI.

Ejemplo DRY
--------------------------------------------

Archivo: DRY.py

Este ejemplo muestra el principio DRY, que significa no repetirse.
La función calcula diferentes áreas sin repetir la lógica en varias funciones. En lugar de escribir múltiples funciones similares, se utiliza una sola función.
Así evitamos la duplicación y el código es más fácil de mantener, entender y menos propenso a errores.

Ejemplo KISS
--------------------------------------------

Archivo: KISS.py

Este ejemplo muestra el principio KISS, que significa mantener el código simple.
La función evalúa si un número es par de forma directa, evitando estructuras innecesarias.
Esto hace que el código sea más claro, fácil de leer y de entender.

Ejemplo YAGNI
--------------------------------------------

Archivo: YAGNI.py

Este ejemplo muestra el principio YAGNI, que significa no implementar cosas que no se necesitan.
Se crea una clase simple de usuario con solo lo necesario, sin agregar funciones o atributos innecesarios.
Esto evita complejidad y facilita el desarrollo.
